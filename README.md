# Smart Transport Recommendation System

## Overview
A web-based Smart Transport Recommendation System that recommends optimal travel modes based on time, cost, and safety preferences.

## Features
- Real-time transport recommendation
- Route visualization
- API integration
- Booking redirection
- User authentication
- Emergency support module
- Trip history management

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Functionalities
- Geocoding and routing API integration
- Recommendation engine
- Distance and travel time calculation
- Database management
- User login system

## Future Enhancements
- Mobile application deployment
- AI-based recommendations
- Real-time traffic integration
- Advanced personalization
